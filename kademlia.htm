Instructions for connecting to Kademlia/eDonkey

On your Windows system download and install the eMule software from the eMule Project

On OSX or Linux use aMule located at aMule Project

With your client installed, add my server address 10.1.1.1 port 4661 or bootstrap your kademlia network using the address 10.1.1.1 port 41859

Instructions for connecting to Freenet

On your Windows, OSX, or Linux system, download and install the Tor software from Tor project download or the local mirror

With tor installed, edit your torrc file to include the following line:

bridge 192.168.0.1:443 A628B7B43445B740E2BB0C0AAE08958D72136FB5

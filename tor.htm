Instructions for connecting to Freenet

On your Windows, OSX, or Linux system, download and install the Tor software from Tor project download

With tor installed, edit your torrc file to include the following line:

bridge 10.1.1.1:443 4352e58420e68f5e40bf7c74faddccd9d1349413
